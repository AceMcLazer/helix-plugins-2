Your central repository for Helix plugins

Take a peek at the officially maintained library of plugins. Each one is approved by a human, so you won't get any nasty surprises.

Browse plugins Learn more

  • Browse

    Take a look at the available plugins and see what suits your needs.

  • Download

    Once you've found a plugin you like, you can download it directly from the repository without having to fuss with Git.

  • Install

    Simply drag the contents of the downloaded zip file into your schema's plugin folder and restart.

  • Create

    Write a plugin to share with the community. Get creative!

  • Submit

    Finished polishing off your plugin? Submit a pull request to the plugin repository to add or update your creation.

  • Share

    Once approved, send the link to your friends, or share it on the Discord server.
