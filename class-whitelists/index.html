Class Whitelists

by wowm0d

  • Description
  • Commands
  • License

Allows classes to be obtainable with whitelists. Add the below funtion to any class you wish to restrict to whitelists.

function CLASS:CanSwitchTo(client) return client:HasClassWhitelist(self.index) end

Players can use these commands to interact with this plugin

/PlyClassWhitelist

Allows someone to change to a specific class within a faction.

/PlyUnClassWhitelist

Disallows someone to change to a specific class within a faction.
