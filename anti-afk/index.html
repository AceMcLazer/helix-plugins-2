Anti-AFK

by Gr4Ss

  • Description
  • Configs
  • License

Stops AFK players from earning wages and kicks when server is full.

Servers can customize usage of this plugin with these settings

AFK Time (default: 300)

The amount of seconds it takes for someone to be flagged as AFK.
