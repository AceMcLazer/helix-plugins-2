Overwatch Rank Manager

by Gary Tate

  • Description
  • Commands
  • License

Allows Overwatch to manage ranks.

Players can use these commands to interact with this plugin

/RankDemote

Demote an active Overwatch functionary.

/RankPromote

Promote an active Overwatch functionary.
